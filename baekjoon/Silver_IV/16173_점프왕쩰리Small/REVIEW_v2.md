**개선할 점**

1. **여전히 대각선 이동**
   - 왜? 48-50번 줄에서 for문은 2번 도는데, next_row와 next_col 계산이 동일함
   - `next_row = row + move`, `next_col = col + move` → 둘 다 같은 값
   - 오른쪽: `(row, col + move)`, 아래: `(row + move, col)` 따로 계산해야 함

2. **경계 조건 오류**
   - 왜? 52번 줄 `next_row > 0 && next_col > 0`은 0번 인덱스 제외
   - `next_row >= 0 && next_col >= 0`이어야 함
   - `next_row < N - 1`도 잘못됨 → `next_row < N`이어야 함 (N-1도 유효한 인덱스)

3. **방문 체크 위치 오류**
   - 왜? 54번 줄에서 dfs 호출 **후**에 방문 체크
   - dfs 호출 **전**에 체크해야 함 (이미 40번 줄에서 하고 있긴 함)

4. **Stack 미사용**
   - 왜? Stack을 생성하고 push만 하고 사용 안 함
   - 재귀로 풀거면 Stack 불필요, 삭제하면 깔끔
