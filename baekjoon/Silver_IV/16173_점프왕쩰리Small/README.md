# 백준 16173번: 점프왕 쩰리 (Small)

| 항목 | 내용 |
|------|------|
| 티어 | Silver IV |
| 링크 | https://www.acmicpc.net/problem/16173 |
| 시간 | 1초 |
| 메모리 | 128MB |

## 문제
'쩰리'는 점프하는 것을 좋아하는 젤리다. 단순히 점프하는 것에 지루함을 느낀 '쩰리'는 새로운 점프 게임을 해보고 싶어 한다.

새로운 점프 게임의 조건은 다음과 같다:
- '쩰리'는 가로와 세로의 칸 수가 같은 정사각형의 구역 내부에서만 움직일 수 있다.
- '쩰리'가 정사각형 구역의 외부로 나가는 경우엔 게임에서 패배한다.
- '쩰리'의 출발점은 항상 가장 왼쪽, 가장 위의 칸이다.
- '쩰리'가 이동 가능한 방향은 **오른쪽과 아래** 뿐이다.
- '쩰리'가 가장 오른쪽, 가장 아래 칸에 도달하면 승리한다.
- '쩰리'가 한 번에 이동할 수 있는 칸의 수는 현재 밟고 있는 칸에 쓰여 있는 수 만큼이다.

## 입력
첫째 줄에 게임 구역의 크기 N이 주어진다. (2 ≤ N ≤ 3)

둘째 줄부터 게임판의 구역이 주어진다. 승리 지점(오른쪽 맨 아래 칸)에는 -1이 쓰여있고, 나머지 칸에는 0 이상 100 이하의 정수가 쓰여있다.

## 출력
'쩰리'가 게임에서 승리할 수 있으면 "HaruHaru", 승리할 수 없으면 "Hing"을 출력한다.

## 예제
### 입력 1
```
3
1 1 10
1 5 1
2 2 -1
```
### 출력 1
```
HaruHaru
```

### 입력 2
```
3
2 2 1
2 2 2
1 2 -1
```
### 출력 2
```
Hing
```

## 힌트
- (0, 0)에서 시작하여 (N-1, N-1)에 도달할 수 있는지 판단
- 이동 방향: 오른쪽, 아래 (2방향)
- 알고리즘: DFS 또는 BFS
